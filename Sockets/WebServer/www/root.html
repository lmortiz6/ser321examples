You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /anagram -- checks if input string1 and string2 are anagrams, ex: /anagram?string1=example1&string2=1example
  • /shift -- returns string with all characters shifted by amount, ex: /shift?string=example&amount=1
